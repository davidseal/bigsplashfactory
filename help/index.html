Bulk Product Descriptions - User Documentation

Create a new bulk product description

  • 1. Navigate to the app by going to the "Apps" section of your Shopify admin, or by searching for "Bulk Product Description" in the search bar.
  • 2. Select the "Add Bulk Product Description" button in the top right corner of the index page
  • 3. Enter the required information for creating a bulk description
  • 4. Select "Save" to create the new bulk description and apply it to all selected products

How is content added to my existing product descriptions?

The content of the bulk product description is added directly to existing product descriptions. If you navigate to a selected product and view it's description, you will be able to see the content added by the app. The content is wrapped in exclusive html tags used by the app for allowing it to easily update and delete content. You can update or delete this content using the app interface. Updating or deleting the content directly in the product description is not encouraged without first deleting the original bulk description
